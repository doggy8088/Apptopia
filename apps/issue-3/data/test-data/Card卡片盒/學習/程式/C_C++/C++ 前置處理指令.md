# 前置指令

前置指令就是，通知C++編譯器，要執行些什麼，以及設定些什麼的命令
以 **\#** 開頭這樣

## C++預設前置指令

## include

很重要的指令，就是要求引入[[標頭檔的作用|標頭檔]]
有分成 ** ""** 、 ** <> ** 兩種
- "" : 是在當前程式目錄找，沒有再去系統目錄找
- <>: 就是直接去系統目錄裡找(就是去編譯器安裝的系統目錄，大部分在C槽)

## define

define 是一個定義常數的方法，
還可以定義 define函式，這樣整個C++專案都能用
取消方式是 undef
