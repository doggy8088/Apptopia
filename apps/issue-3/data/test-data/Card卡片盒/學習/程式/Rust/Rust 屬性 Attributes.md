# 概念

這在[[Rust 權威指南]]並沒有提到，但是在Rust程式碼常常出現以`#[...]`開頭
可以理解成是對Rust編譯器的溝通，只要標上它，就能對程式碼加上**功能**
編譯就會去實現，
例如
- `#[derive(Debug)]`
	讓編譯器可以運行`{:?}`的Debug狀態
- `#[test]`
	可以使用`cargo test`函式
- `#[cfg(...)]`
   用於條件編譯  (test) 就是測試時編譯，target_os = "linux"就是Linux時編譯進去
- `#[allow(...)]` 和 `#[warn(...)]`
	用於控制程式碼警告 例如`#[allow(dead_code)]`就是告訴編譯器別跳警告
