# Apps Directory

This directory contains all applications developed for Apptopia issues.

## Structure

Each application is organized as follows:

```
apps/
└── issue-N/          # N is the issue number (e.g., issue-1, issue-42)
    ├── README.md
    ├── src/
    ├── tests/
    └── ...
```

## Guidelines

Please refer to [AGENTS.md](../AGENTS.md) for complete development guidelines.

## Current Apps

- `example/` - Example directory structure (not a real app)
- `issue-2/` - AI Pixel Renderer CLI (JSON color matrix to GIF/PNG)
- `issue-17/` - Kubernetes 升級驗證器（K8s Upgrade Validator）

Add your app here following the naming convention `issue-N/` where N is your issue number.
